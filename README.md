# Бот для тренировок

Бот помогает составлять тренировки, отслеживать прогресс и упрощает процесс занятий спортом.
Идея проекта возникла из личной потребности: хотелось иметь простой инструмент, который подсказывает упражнения, считает подходы и со временем может давать персонализированные рекомендации.

Технологии:
-Python
-aiogram 3

Текущий функционал:
- Можно выбрать вариант тренировки (день ног/день рук/кардио)
- Выдает 5 рандомных упражнений из списка + 1 случайное кардио в конце, есть возможность перевыбрать упражнения случайным образом
- Считает число подходов, которые можно убавлять, чтобы не забыть сколько сделал, или наоборот прибавить, если сил много ;)

План по развитию:
- Добавить анкету в начале, чтобы учесть уровень физ.подготовки, цель (похудение/поддержание/набор), количество подходов и необходимых упражнений за тренировку
- Учитывать вес, рост, КБЖУ, % жира и мышц в теле, а затем строить графики по тенденции
- На каждое упражнение выводить/менять рабочий вес, чтобы не держать это в голове
- Связать бота с БД для учета разных характеристик под каждого пользовател
- Для женщин: учет цикла, засчет этого рекомендации по нагрузке и питанию. Возможно, учет данных из приложений для отслеживания цикла
- Привязать ИИ, который будет поддерживать и давать рекомендации
- И, естественно, деплой на сервер :)
